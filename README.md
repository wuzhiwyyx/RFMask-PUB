# RFMask
RFMask: A Simple Baseline for Human Silhouette Segmentation with Radio Signals

## Install

```
pip install -r requirements.txt
```

## Usage

<details>
<summary>Training</summary>

```shell
python tools/train.py configs/config.yaml
```

</details>
